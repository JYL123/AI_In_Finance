Install an Express app:
```
npm install -g express-generator // install generator
or
yarn global add express-generator

express react-backend // create express app

cd react-backend && npm install // install dependencies

DEBUG=react-backend:* npm start // run the app

// make sure to install dependencies
cd react-backend
npm install
```
With MongoDB (on terminal):
```
Start mongo server: mongod
Use mongo db: mongo
Show databases: show dbs
Switch to database: use mydb
Show collections: show collections
Display all data in the collection: db.stocks.find().pretty() (stocks is the name of the collection)
```
